# 🌍 Global Climatic Change Dashboard

An interactive data visualization dashboard built using Python, Dash, and Plotly to analyze global climate trends including temperature, precipitation, and CO₂ emissions.

---

## 🚀 Overview

This project visualizes climate change data across multiple countries and years. It helps users explore patterns, trends, and relationships between key climate factors through interactive graphs and filters.

---

## ✨ Features

- 🌡️ Temperature Analysis (Annual & Monthly)
- 💧 Rainfall / Precipitation Trends
- 🏭 CO₂ Emissions Growth Analysis
- 🌍 Choropleth World Map
- 📊 Correlation Heatmap
- 📈 Trend Analysis (Line Graphs)
- 📊 Distribution (Histogram, Donut Charts)
- 🔍 Scatter Plot (Temperature vs CO₂)
- 🔄 Animated Bubble Chart
- 📋 Data Table (Aggregated & Year-wise)
- 🎛️ Filters:
  - Country Selection
  - Year Range Slider
  - Light/Dark Theme Toggle

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas
- Dash
- Plotly
- Dash Table
